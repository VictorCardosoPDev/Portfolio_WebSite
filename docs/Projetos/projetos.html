Lírico Estúdio

Descrição: Projeto de site desenvolvido para o estúdio de tatuagem Lírico Estúdio. O site transmite a identidade visual da marca com foco na apresentação dos tatuadores, portfólios e informações de contato.

Funcionalidades: Exibição dos profissionais com descrição e portfólio individual, informações sobre o estúdio, formulário de contato e responsividade para dispositivos móveis e um modelo em 3D do corpo humano para simular uma tatuagem no corpo.

Tecnologias Utilizadas: HTML, CSS e JavaScript.

Angular Project: Brazilian Food Store

Descrição: Aplicação interativa desenvolvida com Angular para simular um sistema de carrinho de compras. A interface é moderna e funcional, oferecendo uma experiência fluida para o usuário ao visualizar produtos e gerenciar pedidos.

Funcionalidades: Adição e remoção de itens, alteração de quantidade, cálculo automático do valor total por item e do pedido, manipulação de dados com serviços e comunicação entre componentes.

Tecnologias Utilizadas: Angular, TypeScript, HTML, CSS.
