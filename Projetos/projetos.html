Lírico Studio

Projeto de site para um estúdio de tatuagem, desenvolvido com HTML, CSS e JavaScript, com foco em engajamento de clientes. Inclui um simulador de tatuagens, que utiliza um modelo 3D interativo de corpo humano para visualizar tatuagens personalizadas. O simulador é alimentado por Python e bibliotecas como Three.js, GLTFLoader e OrbitControls, rodando em servidor local que possibilita o usuário fazer upload de uma imagem que deseja tatuar.
